### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`

### 三、功能描述
系统分为（身份）：普通用户、企业、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 招聘信息
5. 招聘新闻
6. 投诉建议
7. 后台管理
8. 个人中心

**企业功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 招聘信息管理
6. 人才库管理
7. 简历投递管理
8. 面试邀请管理
9. 求职咨询管理
10. 企业回复管理
11. 招聘新闻管理




**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 学生管理
4. 企业管理
5. 招聘信息管理
6. 行业类型管理
7. 人才库管理
8. 简历投递管理
9. 面试邀请管理
10. 求职咨询管理
11. 企业回复管理
12. 招聘新闻管理
13. 投诉建议
14. 系统管理（轮播图管理）
